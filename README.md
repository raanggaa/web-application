# web-application

// Deskripsi Proyek :
Aplikasi ini adalah simulasi sebuah sistem manajemen sebuah lagu, di mana user dapat menambahkan, mengedit, menghapus, dan melihat data-data lagu yang telah dibuat. User dapat menyimpan data-data lagu yang mereka dengarkan atau sukai, seperti apa judulnya, siapa penyanyinya, berapa lama durasi lagunya, dan apa genre lagunya.

// Alasan Memilih Tema
Saya memilih tema ini karena saya suka buat mendengarkan musik. Saya sering penasaran dengan genre lagu-lagu yang saya dengarkan, jadi saya membuat sistem manajemen sebuah lagu yang bisa menyimpan data-data seperti judul, artis, genre, dan album dari lagu-lagu yang saya dengarkan.
